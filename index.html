Accessible Linguistics

While learning to code I was amazed to see all the great resources online for learning programming languages: freely accessible documentation, lessons, and tutorials with interactive code playgrounds.

Wouldn't it be great if we had similar resources for learning the more complex and less documented (resource limited) human languages?

I created the brand LingDocs to publish these kinds of resources.

LingDocs aims to:

  • Use modern web tech to deliver rare language materials in a way that's smooth, accesible, and user friendly.
  • Share solid linguistic information in the simplest terms possible, making it accessible for non-linguists and average language learners.
